# Support

Use the right channel so maintainers and contributors can respond properly.

## Questions and Ideas

Use GitHub Discussions for:

- General questions.
- Product ideas.
- Community feedback.
- Zakat, welfare, organization, and tazkiyah workflow suggestions.
- Open-ended design discussion.

## Bugs and Specific Work

Use GitHub Issues for:

- Documentation errors.
- Specific feature requests.
- Clear implementation tasks.
- Reproducible bugs once code exists.

## Private or Sensitive Matters

Do not post private hardship, zakat, family, legal, financial, or security details publicly.

For sensitive matters, ask maintainers for a private contact path without sharing details in public.

## Religious, Legal, Tax, Medical, or Financial Advice

Taqdimah can support workflows, records, discovery, and coordination. It does not replace qualified scholars, lawyers, accountants, doctors, or financial advisors.
